Add new Todo: click 'Add New Todo', type in new Todo and press Enter.
[Note: This input box can be hidden by clicking the Edit button on the top-right corner.]

Delete Todo: click the Trash bin button with red background.

Highlight Todo: click the up arrow button with green background.
[Note: highlighted Todo will be placed as the first item in the list]

Move Todo to bottom: click the down arrow button with black background.
[Note: works for all Todo's, including highlighted Todo; highlighted Todo will become normal Todo]

Line-through Todo: click anywhere else on the Todo will apply a line-through on the Todo. click again to remove this effect.

At version 1.0, the four Todos of workout, reading, coding, and diet plan are set as default Todo, which are the daily routines of author sean wang.
